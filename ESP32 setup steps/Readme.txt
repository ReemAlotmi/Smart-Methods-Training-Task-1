1- Go to https://www.arduino.cc/ --> software --> download Arduino ide and install it

2- Go to https://docs.espressif.com/projects/arduino-esp32/  --> getting started --> installing --> copy the stable release link

3- open Arduino ide and go to file --> references --> paste the link to the Additional Boards Manager URLs box --> OK

4- Tools --> Board: "Arduino Uno" --> Boards Manager --> search for esp32 --> install 

5- now the esp32 package is installed , you can find the boards in Tools --> Board: "Arduino Uno" --> ESP32 Arduino
